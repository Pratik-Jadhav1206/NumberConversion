Decimal to Hexadecimal Conversion

Decimal Number System

The decimal number system uses base 10 and digits from 0 to 9. Each position represents powers of 10.

Example:

(345)₁₀ = (3 × 10²) + (4 × 10¹) + (5 × 10⁰)

Hexadecimal Number System

The hexadecimal system uses base 16 and includes digits: 0–9 and A–F.

Decimal Hex
10A
11B

15F

Conversion Method

Divide the decimal number repeatedly by 16 and record the remainders.

Then read the remainders from bottom to top to get the hexadecimal value.
