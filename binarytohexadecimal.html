Binary and Hexadecimal Number System

Binary Number System

The binary number system uses base 2. It contains only two digits: 0 and 1. Binary numbers are used in digital computers because electronic circuits have two states: ON (1) and OFF (0).

Each digit in binary is called a bit. Binary numbers follow a positional system where each digit represents a power of 2.

Example Binary Number: (1011)₂

(1 × 2³) + (0 × 2²) + (1 × 2¹) + (1 × 2⁰)

Hexadecimal Number System

The hexadecimal number system uses base 16. It contains sixteen symbols.

Digits: 0–9 and letters A–F.

The letters represent values:

  • A = 10
  • B = 11
  • C = 12
  • D = 13
  • E = 14
  • F = 15
Example Hex Number: (2F)₁₆

(2 × 16¹) + (15 × 16⁰)

How to Convert Binary to Hexadecimal

Binary numbers are converted into hexadecimal by grouping digits into sets of four bits starting from the right side.

Steps

  • Step 1: Write the binary number.
  • Step 2: Group digits into sets of 4.
  • Step 3: Convert each group into hexadecimal.
  • Step 4: Combine the results.

Example Conversion

Convert (10101111)₂

Step 1: Group digits 1010 1111

Step 2: Convert each group 1010 = A 1111 = F

Final Result:
(10101111)₂ = (AF)₁₆
