Octal to Hexadecimal

Octal Number System

The octal number system is a base-8 number system. It uses eight digits: 0,1,2,3,4,5,6,7. Each digit represents a power of 8.

Example:

(325)8 = 3×8² + 2×8¹ + 5×8⁰

Hexadecimal Number System

The hexadecimal system is a base-16 number system used in computer programming and digital electronics.

It uses digits:

  • 0-9
  • A = 10
  • B = 11
  • C = 12
  • D = 13
  • E = 14
  • F = 15

Why Convert Octal to Hexadecimal?

  • Used in low-level programming
  • Used in memory addressing
  • Useful in digital system design

Method to Convert Octal to Hexadecimal

Octal cannot be directly converted to hexadecimal easily. So we convert through binary.

Step 1

Convert each octal digit to 3-bit binary.

Step 2

Group binary digits into groups of 4 from the right.

Step 3

Convert each 4-bit group into hexadecimal.

Example 1

Convert (17)8 to Hexadecimal

1 → 001 7 → 111 Binary = 001111 Group → 0011 11 0011 = 3 001111 = F
Hexadecimal = F

Example 2

Convert (345)8

3 → 011 4 → 100 5 → 101 Binary = 011100101 Group = 0001 1100 101 Hex = 1C5
