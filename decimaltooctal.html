Decimal to Octal

Decimal Number System

The decimal number system uses base 10 and digits from 0 to 9. Each position in a decimal number represents a power of 10.

Example:

(345)₁₀ = (3 × 10²) + (4 × 10¹) + (5 × 10⁰)

Decimal numbers are commonly used in everyday life but computers often convert them into other number systems such as binary, octal, and hexadecimal.

Octal Number System

The octal number system uses base 8 and digits from 0 to 7. Each digit position represents a power of 8.

Example:

(325)₈ = (3 × 8²) + (2 × 8¹) + (5 × 8⁰)

= 192 + 16 + 5 = 213₁₀

Decimal to Octal Conversion Method

To convert a decimal number to octal we use repeated division by 8.

  1. Divide the decimal number by 8.
  2. Write the remainder.
  3. Divide the quotient again by 8.
  4. Repeat until the quotient becomes zero.
  5. Read remainders from bottom to top.

Example

Convert decimal number 83 into octal.

Division Quotient Remainder
83 ÷ 8 10 3
10 ÷ 8 1 2
1 ÷ 8 0 1

Reading remainders from bottom to top gives the octal result:

123₈

Advantages of Octal System

  • Short representation of binary numbers.
  • Easy conversion between binary and octal.
  • Used in low level programming.
  • Helpful in digital electronics.

Applications of Octal Numbers

  • Computer memory addressing
  • Microprocessor programming
  • Digital electronics
  • UNIX file permissions
  • Binary number representation
