Hexadecimal to Decimal Conversion

Hexadecimal Number System

The hexadecimal number system uses base 16 and consists of 16 symbols: 0–9 and A–F. It is commonly used in computing because it provides a short representation of binary numbers.

Hex Digit Decimal Value
A10
B11
C12
D13
E14
F15

Hexadecimal to Decimal Conversion Method

To convert a hexadecimal number into decimal, each digit is multiplied by the power of 16 according to its position. The rightmost digit has power 16⁰, the next digit has power 16¹, and so on.

Example

Convert (2F)16 into decimal.

F = 15

(2F)16 = (2 × 16¹) + (15 × 16⁰)

= 32 + 15

Decimal Result = 4710

Applications of Hexadecimal Numbers

  • Computer memory addresses
  • Machine level programming
  • HTML and CSS color codes
  • Debugging software programs
