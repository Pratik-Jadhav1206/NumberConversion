Binary and Octal Number System

Binary System

The binary number system uses base 2 and contains only two digits: 0 and 1. Binary numbers are widely used in computers and digital electronics because electronic circuits have two states: ON (1) and OFF (0).

Example Binary Number: (1011)₂

(1 × 2³) + (0 × 2²) + (1 × 2¹) + (1 × 2⁰)

Octal System

The octal number system uses base 8 and contains eight digits: 0,1,2,3,4,5,6,7.

Example Octal Number: (345)₈

(3 × 8²) + (4 × 8¹) + (5 × 8⁰)

Binary to Octal Conversion

  • Step 1: Write the binary number.
  • Step 2: Group digits into sets of 3 from the right.
  • Step 3: Convert each group into octal.
  • Step 4: Combine the octal digits.
Example: Convert (101011)₂

Step 1: Group digits 101 011

Step 2: Convert groups
101 = 5 011 = 3

Result = (53)₈
