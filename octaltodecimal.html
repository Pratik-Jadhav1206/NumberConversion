Octal to Decimal Conversion

Octal Number System

The octal number system is a base-8 number system. It uses eight digits: 0, 1, 2, 3, 4, 5, 6 and 7. Each position in an octal number represents a power of 8.

Example:

(325)8 = 3 × 8² + 2 × 8¹ + 5 × 8⁰

Decimal Number System

The decimal number system is the most commonly used number system. It is a base-10 system that uses digits from 0 to 9.

Each position represents powers of 10.

Why Convert Octal to Decimal?

  • Computers sometimes store values in octal format.
  • Humans understand decimal numbers more easily.
  • Conversion helps interpret machine values.

Steps to Convert Octal to Decimal

  • Step 1: Write the octal number.
  • Step 2: Multiply each digit by the power of 8 based on its position.
  • Step 3: Add all the results.

Example 1

Convert (157)8 to decimal

1 × 8² = 64
5 × 8¹ = 40
7 × 8⁰ = 7

Decimal = 64 + 40 + 7
Result = 111

Example 2

Convert (243)8 to decimal

2 × 8² = 128
4 × 8¹ = 32
3 × 8⁰ = 3

Decimal = 128 + 32 + 3
Result = 163
